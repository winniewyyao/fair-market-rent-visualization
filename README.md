# Visualize Fair Market Rent Dataset

A data analysis and visualization project for J221 final project.

All data cleaning and analysis were done in <a href="https://github.com/winniewyyao/fair-market-rent-data-analysis/blob/main/J221-final.ipynb">this Jupyter Notebook</a> before exporting the curated CSV files to Datawrapper for visualization. Check out the final presentation <a href="https://winniewyyao.github.io/fair-market-rent-visualization/">here</a>.
